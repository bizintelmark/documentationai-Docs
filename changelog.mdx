## Upgrading Your Instance

Follow these steps to upgrade Biz Intel Marketing safely.

<Steps>
  <Step title="Backup Data" icon="database">

    Export your current data using the dashboard export feature.

    ```
    curl -X POST https://api.example.com/v1/export \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -d '{"campaigns": true, "analytics": true}'
    ```

  </Step>
  <Step title="Update SDK" icon="package">

    <Tabs>
      <Tab title="npm" icon="nodejs">
````bash
npm install @bizintel/marketing@latest
````
      </Tab>
      <Tab title="Python" icon="python">
````bash
pip install bizintel-marketing==1.2.0
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Execute database migrations if using self-hosted deployment.

    ```
    bizintel migrate --env production
    ```

  </Step>
  <Step title="Verify" icon="check-circle">

    Test key endpoints and dashboards.

  </Step>
</Steps>

## Migration Notes for Breaking Changes

For v1.0.0 users, update your API calls:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Old (beta)
  const data = await fetch('https://api.example.com/beta/campaigns');

  // New (v1)
  const data = await fetch('https://api.example.com/v1/campaigns', {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });
  ```
  ```python
  import requests

  # Old (beta)
  response = requests.get('https://api.example.com/beta/campaigns')

  # New (v1)
  response = requests.get('https://api.example.com/v1/campaigns', headers={
      'Authorization': f'Bearer {YOUR_API_KEY}'
  })
  ```
</CodeGroup>
