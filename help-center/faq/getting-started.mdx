## Overview

To start using Biz Intel Marketing, create an account and set up your first project.

## Steps

1. Go to `https://dashboard.bizintelmarketing.com`.
2. Click "Sign Up" and complete registration.
3. Verify your email address.
4. Log in and click "Create Project".
5. Name your project and select a template.

You’re now ready to explore features and add content.
